# 会议纪要、研讨会自动生成文档工具

## 原需求描述

依托语音识别、语言处理等人工智能技术，实现工程类会议的语音实时转文字，自动提取会议核心议题、讨论内容、决议事项、责任分工、时间要求等关键信息；对提取的信息进行结构化梳理、逻辑化排版，生成标准化会议纪要初稿，支持人工编辑，完成后可直接同步至项目管理相关台账。

---

## 页面结构

```
- 会议工具首页
  - 功能入口
  - 历史会议

- 会议记录页
  - 实时转写
  - 内容展示

- 纪要生成页
  - 内容编辑
  - 预览导出

- 台账同步页
  - 项目关联
  - 同步记录
```

## 功能列表

### 1. 语音实时转写

- 实时语音识别
- 多发言人区分
- 转写内容展示：
  ```
  [转写内容]

  发言人1 [张三]：
  关于XX路段的施工方案，建议采用...

  发言人2 [李四]：
  同意张经理的意见，需要补充...

  发言人3 [王五]：
  资金方面需要再评估...
  ```

### 2. 关键信息提取

- 核心议题识别
- 决议事项提取
- 责任分工
- 时间要求

### 3. 结构化纪要生成

- 模板化输出：
  ```
  [会议纪要]

  会议名称：XX工程进度协调会
  会议时间：2024-01-15 14:00-16:00
  参会人员：张三、李四、王五...

  一、会议议题
  ├─ XX路段施工方案确定
  └─ 资金安排调整

  二、讨论内容
  [详见转写记录]

  三、决议事项
  ├─ 原则同意采用方案A
  ├─ 追加预算¥500万
  └─ 3月底前完成招标

  四、任务分工
  ├─ 张三：负责方案细化
  ├─ 李四：负责资金申请
  └─ 王五：负责招标准备

  五、时间要求
  └─ 2024-03-31前完成
  ```

### 4. 人工编辑

- 在线编辑
- 修改痕迹保留
- 版本管理

### 5. 台账同步

- 项目关联
- 自动同步至项目管理
- 同步状态跟踪

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| meeting_id | string | 会议ID |
| transcript | text | 转写内容 |
| speaker_id | string | 发言人 |
| extracted_info | object | 提取信息 |
| generated_doc | string | 生成文档 |
| project_id | string | 项目ID |
